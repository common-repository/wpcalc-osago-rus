Калькулятор ОСАГО по тарифам России

== Description ==

WordPress плагин калькулятора ОСАГО для расчета страхового платежа на территории России. Плагин предоставляет собой возможность вставки калькулятора ОСАГО на любой сайт, работающий на CMS Wordpress с помощью шорткода.
Калькулятор рассчитывает страховой тариф по ОСАГО на все транспортные средства по действующим тарифам.

== Installation ==

1. Загрузите `wpcalc-osago-rus` папку в директорию  `/wp-content/plugins/`.
2. Активируйте плагина в разделе "Плагины" в админпанели WordPress.
3. Перейдите на страницу настроек (OSAGO RUS) - в разделе "Настройки" в админпанели Вашего сайта.

Для вставки на страницу или в запись используйте шорткод [wpcosagorus]
